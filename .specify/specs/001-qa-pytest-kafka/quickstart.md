# Quickstart: Kafka BDD Steps Module

## Prerequisites
- Python 3.13
- Kafka broker running and accessible
- Install dependencies via PDM (see monorepo root)
- Install confluent-kafka-python and types-confluent-kafka

## Minimal Example

```python
from qa_pytest_kafka.kafka_steps import KafkaSteps
from qa_pytest_kafka.kafka_configuration import KafkaConfiguration

config = KafkaConfiguration(bootstrap_servers="localhost:9092", group_id="test-group")
steps = KafkaSteps(config)

steps.given.a_kafka_topic("test-events", partitions=3)\
     .when.publishing([
         {"content": {"event": "login", "seq": 1}, "headers": {"source": "web"}, "key": "user-123", "partition": 0},
         {"content": {"event": "logout", "seq": 2}, "headers": {"source": "web"}, "key": "user-123", "partition": 1},
     ])\
     .then.the_messages_from_partition_match(0, expected=[...])
```

## Test Execution
- Run tests with `pdm run pytest`
- Allure and HTML reports are generated by default
